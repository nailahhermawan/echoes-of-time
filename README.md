## Tech Stack
- **Engine:** Godot 4
- **Art:** Aseprite
- **Language:** GDScript

## Cara Menjalankan
1. Install [Godot 4](https://godotengine.org/download/) (versi Standard)
2. Clone repo ini
3. Buka Godot → Import → arahkan ke folder repo
4. Tekan `F5` untuk run
